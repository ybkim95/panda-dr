## panda-dr
Domain Randomization (DR) with Panda Arm in PyREP Environment

<br>

## Set-up
Place panda-dr folder into PyRep and run panda-dr.py 

(Ex) ~/PyRep/panda-dr$ python3 panda-dr.py
